# Getting Started
**Install:**
- .NET 6, EF6
- NodeJs 16.16
- Angular 16
- Sql lite

### Create database
Go to API folder, run command:
```
dotnet ef database update
```

### Running the Application
Go to the API folder, run command:
```
dotnet run
```

Go to the client folder, run command:
```
npm install
```
Then:
```
npm start
```
